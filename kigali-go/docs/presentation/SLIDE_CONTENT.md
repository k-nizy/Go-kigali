## **SLIDE 2: Problem Statement**
**Speaker: Team Member 1** (1 min)

**Key Statistics:**
- "67% of Kigali residents rely on public transport daily" (Rwanda Transport Authority, 2023)
- "Average wait time: 35-45 minutes during peak hours" (World Bank, 2022)
- "42% of passengers report fare overcharging" (Rwanda Consumer Rights, 2023)

**Main Problems:**
1. No real-time transport information
2. Fare transparency issues  
3. Limited safety reporting
4. Inefficient route planning

**Who's Affected:** Daily commuters, visitors, women travelers, low-income workers

## **SLIDE 4: KigaliGo Solution**
**Speaker: Team Member 2** (1 min)

**Value Proposition:**
"Empowering Kigali passengers with real-time transport intelligence, transparent pricing, and safety-first planning"

**Our Approach:**
- ✓ Multi-modal integration (buses, taxis, motos)
- ✓ Passenger-centric design
- ✓ Built specifically for Kigali
- ✓ Progressive Web App (no download needed)
- ✓ Offline-capable

**Technology:** React PWA + Flask API + PostgreSQL + PostGIS

## **SLIDE 7: Technical Architecture**
**Speaker: Team Member 4** (45 sec)

**Architecture:**
```
User Phone ←→ React PWA ←→ Flask API ←→ PostgreSQL+PostGIS
```

**Key Technical Features:**
- Progressive Web App (no app store needed)
- Real-time data processing (2-3 sec)
