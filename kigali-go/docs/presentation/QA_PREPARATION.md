# Q&A Preparation Guide
## Anticipated Questions & Answers

### **Q1: How do you get real-time vehicle location data?**
**Responder: Member 4**

Currently using:
1. Integration with transport operators who share GPS data
2. Crowdsourced data from driver companion app
3. Predictive modeling based on schedules

Future: Direct API access to fleet management systems

### **Q2: Data privacy and security?**
**Responder: Member 1**

- HTTPS encryption for all data
- No personal data without consent
- Anonymized analytics
- Rwanda Data Protection law compliant
- User can delete account anytime
